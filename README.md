# Python-Turtle-Art

This program uses Turle to create something artistic.  Several turtles are intiated at different parts of the screen (window created) and they move along leaving a rainbow path behind them.  The time can be changed so that the turtles either fill up the whole screen and ten some or they can leave parts of the canvas emtpy.  When they are done, the "Art" that remains is saved as an .eps file.
